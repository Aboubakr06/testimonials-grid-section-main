### What I learned

```css
@media (min-width:47em) {

    .cards {
        grid-template-columns: repeat(4, 1fr);
        grid-template-rows:  1fr;
        grid-template-areas: 
            'a a b e'
            'c d d e'
                    ;
    }

    .cards> :nth-child(1) {
        grid-area: a;
    }

    .cards> :nth-child(2) {
        grid-area: b;
    }

    .cards> :nth-child(3) {
        grid-area: c;
    }

    .cards> :nth-child(4) {
        grid-area: d;
    }

    .cards> :nth-child(5) {
        grid-area: e;
    }
   
}
```
